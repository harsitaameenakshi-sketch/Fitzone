# 🏋️ FitZone

A modern **Full Stack MERN Fitness Management Platform** built using **React, Node.js, Express and MongoDB Atlas**.

---

## 🌐 Live Demo

🔗 https://fitzone-delta.vercel.app

---

## ✨ Features

- 🏋️ Modern Responsive Landing Page
- 📝 Contact Form
- 🎯 Free Trial Booking
- 💪 Membership Registration
- ☁️ MongoDB Atlas Integration
- 🚀 Express REST APIs
- 🔥 Toast Notifications
- ⏳ Loading Buttons
- ✅ Form Validation
- 🌍 Fully Deployed Application

---

## 🛠 Tech Stack

### Frontend
- React

- CSS3
- React Icons
- React Hot Toast

### Backend
- Node.js
- Express.js

### Database

- Mongoose

- Render
